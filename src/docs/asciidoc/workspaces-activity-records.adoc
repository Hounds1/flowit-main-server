== 워크스페이스 최근 활동 조회

[cols="1,4",role="endpoint-summary"]
|===
|메서드 |`GET`
|경로 |`/v1/workspaces/{workspaceId}/activity-records`
|API 상태 |[.badge.stability.experimental]#Experimental#
|목적 |워크스페이스 안에서 발생한 작업/멤버십 활동을 최근 N일 범위로 조회합니다.
|인증 |JWT access token이 필요합니다.
|권한 |요청자는 해당 워크스페이스의 활성 멤버여야 합니다.
|기본 조회 조건 |`topic=ALL`, `rangeDays=5`를 기본값으로 사용합니다.
|렌더링 단위 |`items[]`의 레코드 하나가 최근 활동 목록의 한 줄 렌더링 단위입니다.
|표현 방식 |서버는 문장을 강제하지 않고 `actor`, `target`, `action`, `changeCount`, `changedElements` 의미 키를 반환합니다.
|상세 diff |`changes[]`는 단일 `action`에 딸린 상세 변경 목록이며, 기본 목록 렌더링이 아니라 펼침/상세 UI에서 사용합니다.
|===
