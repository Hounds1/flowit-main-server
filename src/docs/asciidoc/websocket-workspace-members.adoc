== 워크스페이스 멤버 WebSocket

[cols="1,4",role="endpoint-summary"]
|===
|상위 문서 |link:websocket-notifications.html[알림 WebSocket]
|구독 destination |``/user/queue/notifications``
|payload |``NotificationAlertResponse``
|도메인 |워크스페이스 멤버십 변경
|역할 |워크스페이스 멤버 가입, 역할 변경, 강제 퇴장, 탈퇴 이벤트를 개인 알림으로 전달합니다.
|주의 |별도 destination이 아니라 공통 알림 큐로 내려오는 워크스페이스 멤버 도메인 payload 세부 계약입니다.
|===

== 발송 케이스

[cols="1,1,4",options="header",role="api-table"]
|===
|알림 유형 |수신자 |설명

|``WORKSPACE_MEMBER_JOINED``
|워크스페이스 구성원
|새 멤버가 워크스페이스에 가입했음을 알립니다.

|``WORKSPACE_MEMBER_ROLE_CHANGED``
|워크스페이스 구성원
|멤버 역할이 변경되었음을 알립니다.

|``WORKSPACE_MEMBER_REMOVED``
|강제 퇴장 후 남아 있는 워크스페이스 구성원
|특정 멤버가 워크스페이스에서 강제 퇴장되었음을 구성원 관점으로 알립니다.

|``WORKSPACE_ACCESS_REVOKED``
|강제 퇴장 당사자
|현재 사용자의 워크스페이스 접근 권한이 회수되었음을 당사자 관점으로 알립니다.

|``WORKSPACE_MEMBER_WITHDRAWN``
|워크스페이스 구성원
|멤버가 스스로 워크스페이스에서 탈퇴했음을 알립니다.
|===

== enum 상수

=== NotificationAlertType

[cols="1,4",options="header",role="api-table"]
|===
|상수 |설명

|``WORKSPACE_MEMBER_JOINED``
|워크스페이스 멤버 가입 알림입니다.

|``WORKSPACE_MEMBER_ROLE_CHANGED``
|워크스페이스 멤버 역할 변경 알림입니다.

|``WORKSPACE_MEMBER_REMOVED``
|구성원 관점의 워크스페이스 멤버 강제 퇴장 알림입니다.

|``WORKSPACE_ACCESS_REVOKED``
|강제 퇴장 당사자 관점의 워크스페이스 접근 권한 회수 알림입니다.

|``WORKSPACE_MEMBER_WITHDRAWN``
|워크스페이스 멤버 자진 탈퇴 알림입니다.
|===

=== NotificationScopeType

[cols="1,4",options="header",role="api-table"]
|===
|상수 |설명

|``WORKSPACE``
|알림 범위가 워크스페이스입니다.
|===

=== NotificationActorType

[cols="1,4",options="header",role="api-table"]
|===
|상수 |설명

|``USER``
|알림을 발생시킨 주체가 사용자입니다.
|===

=== NotificationSubjectType

[cols="1,4",options="header",role="api-table"]
|===
|상수 |설명

|``WORKSPACE_MEMBER``
|알림 대상이 워크스페이스 멤버입니다.
|===

=== NotificationLinkType

[cols="1,4",options="header",role="api-table"]
|===
|상수 |설명

|``WORKSPACE_MEMBERS``
|해당 워크스페이스 멤버 화면으로 이동할 수 있습니다. ``link.workspaceId``가 함께 내려옵니다.

|``NONE``
|이동 가능한 워크스페이스 내부 링크가 없습니다. 강제 퇴장 당사자는 해당 워크스페이스 안으로 이동할 수 없으므로 이 값을 사용합니다.
|===

=== ActivityChangeElement

워크스페이스 멤버 WebSocket의 ``changes[].element``에서 사용하는 상수입니다.

[cols="1,4",options="header",role="api-table"]
|===
|상수 |설명

|``MEMBERSHIP``
|워크스페이스 멤버십 상태가 변경되었음을 의미합니다. 가입, 강제 퇴장, 자진 탈퇴에서 사용됩니다.

|``ROLE``
|워크스페이스 멤버 역할이 변경되었음을 의미합니다. 가입, 역할 변경, 강제 퇴장, 자진 탈퇴에서 사용됩니다.

|``OWNERSHIP_TRANSFER``
|자진 탈퇴 과정에서 워크스페이스 소유권이 다른 멤버에게 승계되었음을 의미합니다.
|===

== payload 해석 기준

[cols="1,1,4",options="header",role="api-table"]
|===
|필드 |의미 |설명

|``scope.id``
|``workspaceId``
|알림이 발생한 워크스페이스 식별자입니다.

|``actor.id``
|``userId``
|알림을 발생시킨 사용자 식별자입니다.

|``subject.id``
|``workspaceMemberId``
|알림 대상 워크스페이스 멤버 식별자입니다. 사용자 식별자가 아닙니다.

|``link.workspaceId``
|``workspaceId`` 또는 ``null``
|클라이언트가 이동할 수 있는 워크스페이스 식별자입니다. ``link.type``이 ``NONE``이면 ``null``입니다.

|``changes[].from``
|변경 전 값
|변경 요소에 따라 문자열 또는 객체가 내려올 수 있습니다. 값이 없으면 ``null``입니다.

|``changes[].to``
|변경 후 값
|변경 요소에 따라 문자열 또는 객체가 내려올 수 있습니다. 값이 없으면 ``null``입니다.
|===

== 클라이언트 처리 기준

[cols="1,4",role="endpoint-summary"]
|===
|문장 조립 |``type``, ``actor``, ``subject``, ``scope``를 기준으로 클라이언트에서 문장을 조립합니다.
|강제 퇴장 분기 |``WORKSPACE_MEMBER_REMOVED``는 남아 있는 구성원에게, ``WORKSPACE_ACCESS_REVOKED``는 강제 퇴장 당사자에게 내려옵니다.
|링크 처리 |``link.type``이 ``NONE``이면 워크스페이스 내부 화면으로 이동시키지 않습니다.
|식별자 처리 |``subject.id``는 ``workspaceMemberId``입니다. 사용자 식별자가 필요한 경우 ``actor.id``와 혼동하지 않아야 합니다.
|상태 동기화 |멤버 목록 화면에 있는 사용자는 알림 수신 후 link:workspaces-members.html[워크스페이스 멤버 목록 조회 API]를 다시 호출해 서버 상태와 동기화할 수 있습니다.
|===
