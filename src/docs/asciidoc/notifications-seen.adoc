== 알림 확인 처리

[cols="1,4",role="endpoint-summary"]
|===
|메서드 |``PATCH``
|경로 |``/v1/notifications/seen``
|API 상태 |[.badge.stability.experimental]#Experimental#
|목적 |현재 사용자의 아직 확인하지 않은 알림을 확인 처리합니다.
|인증 |JWT access token이 필요합니다.
|호출 시점 |알림 오버레이가 열렸음을 클라이언트가 서버에 명시할 때 호출합니다.
|처리 기준 |요청 처리 시각을 ``seenAt``으로 사용합니다. 시간 값은 Unix epoch seconds 기준입니다.
|읽음과의 차이 |확인 처리는 알림을 봤다는 상태이며, 읽음 처리는 별도의 link:notifications-read-all.html[알림 모두 읽음 API]에서 수행합니다.
|===
