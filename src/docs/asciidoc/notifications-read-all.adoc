== 알림 모두 읽음

[cols="1,4",role="endpoint-summary"]
|===
|메서드 |``PATCH``
|경로 |``/v1/notifications/read-all``
|API 상태 |[.badge.stability.experimental]#Experimental#
|목적 |현재 사용자의 읽지 않은 알림을 모두 읽음 처리합니다.
|인증 |JWT access token이 필요합니다.
|처리 기준 |요청 처리 시각을 ``readAt``으로 사용합니다. 시간 값은 Unix epoch seconds 기준입니다.
|확인 처리 |모두 읽음 처리 시 아직 확인하지 않은 알림도 같은 시각으로 확인 처리됩니다.
|성공 응답 |읽음 처리 기준 시각과 이번 요청으로 읽음 처리된 알림 수를 반환합니다.
|===
