== 개요

`error.code` 는 클라이언트가 다국어 처리와 에러 분기에 사용할 수 있는 안정적인 식별자입니다.

서버는 하위 호환성을 깨지 않는 범위에서 `error.code` 값을 변경하지 않습니다. 따라서 클라이언트는 `error.code` 를 i18n key 또는 분기 key로 안전하게 사용하는 것을 권장합니다.

`error.message` 는 서버가 제공하는 기본 메시지입니다. 사용자에게 최종적으로 표시할 문구는 각 클라이언트의 i18n 리소스에서 `error.code` 를 기준으로 결정하는 것을 권장합니다.
